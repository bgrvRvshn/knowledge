# Инструмент командной строки
### Основные команды
```bash
# справка
help
# история
history
# очистка
clear
# выход
exit
```
### Создание директории
```bash
# создание директории
mkdir [dirname]
# пример
mkdir my-app
# несколько диреторий
mkdir -p {dir1,dir2}
# несколько вложенных директорий
mkdir -p my-app/{css,js}
```
